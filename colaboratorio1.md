# GOOGLE COLAB
Google Colab es una plataforma de Python en la nube. Permite programar y ejecutar código desde el navegador sin instalar nada en tu computadora.
## Qué se necesita
Solo dos cosas:
* Una cuenta de Google Account
* Navegador web
## Pasos a seguir
* Entrar a <https://colab.research.google.com>
* Iniciar sesión y crear un nuevo cuaderno
* Configurar especificaciones (como por ejemplo CPU y GPU)
