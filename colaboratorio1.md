# GOOGLE COLAB
Google Colab es una plataforma de Python en la nube. Permite programar y ejecutar código desde el navegador sin instalar nada en tu computadora.
## Qué se necesita
Solo dos cosas:
* Una cuenta de * Google Account *
* Navegador web
## Pasos a seguir
* Entrar a <https://colab.research.google.com>
* Iniciar sesión y crear un nuevo cuaderno
* Configurar especificaciones (como por ejemplo CPU y GPU)
Esto crea una máquina virtual en la nube donde corre el código. También se puede se puede instalar librerías con paquetes si es necesario.
## Guardado
Los notebooks se guardan automáticamente en Google Drive.
Esto sería un resumen sencillo de como funciona Google Colab, con un a pequeña instrucción de uso.
